## Информация

Запрос

Вы можете настроить поведение системы, когда цена вашего лимитного ордера превышает максимальную цену спроса или
минимальную цену предложения. Вы можете запросить конфигурацию, выбрав
[Получить лимит ценового поведения](<Получить лимит ценового поведения.md>).

Spot

- Максимальная цена покупки: min(max(indexprice, indexprice(1 + y%) + 2 - Minute Average Premium), indexprice(1 + z%))
- Минимальная цена продажи: max(min(indexprice, indexprice(1 – y%) + 2 - Minute Average Premium), indexprice(1 – z%))

Futures

- Максимальная цена покупки: min(max(indexprice, markprice(1 + x%)), markprice(1 + y%))
- Минимальная цена продажи: max(min(indexprice, markprice(1 - x%)), markprice(1 - y%))

> Настройка по умолчанию:
>
> - Спот: `modificationEnable` = `false`.
>   - Если цена ордера превышает границу, система отклоняет запрос.
>   - Соответствует поведению цены [Получить лимит ценового поведения](<Получить лимит ценового поведения.md>),
>     где `lpaSpot` = `false`, `lpaPerp` = `true`
>
> - Фьючерсы: `modificationEnable` = `true`.
>
>   - Если цена ордера превышает границу, система автоматически скорректирует цену до ближайшего допустимого значения
>    (т.е. максимальной цены спроса или минимальной цены предложения).
>   - Соответствует поведению цены [Получить лимит ценового поведения](<Получить лимит ценового поведения.md>),
>    где `lpaSpot` = `true`, `lpaPerp` = `false`
>
> - Выбор линейного или обратного значения задаст поведение для всех фьючерсов.

## Конечная точка

`/v5/account/set-limit-px-action`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/account/set-limit-px-action HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1753255927950
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 52
  
  {
      "category": "spot",
      "modifyEnable": true
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/set-limit-px-action"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "category": "spot",
      "modifyEnable": True
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

## Параметры запроса

| Параметр     | Обязательный  | Тип     | Комментарии                                                                                   | По умолчанию |
|--------------|---------------|---------|-----------------------------------------------------------------------------------------------|--------------|
| category     | да            | string  | ***Тип продукта.***<br><br>- `linear`, `inverse`, `spot`                                      | -            |
| modifyEnable | да            | boolean | - `true`: разрешить системе изменять цену ордера<br>- `false`: отклонить ваш запрос на ордер  | -            |
