## Информация

Запрос на ручное погашение займа без конвертации активов

> Информация:
>
> Логика валидации при погашении
>
> Когда указаны `coin` и `amount`, применяются следующие правила в зависимости от `repaymentType`:
>
> - `ALL`: Запрос отклоняется, если amount превышает общую сумму обязательств.
> - `FIXED`: Запрос отклоняется, если amount превышает обязательства с фиксированной ставкой.
> - `FLEXIBLE`: Запрос отклоняется, если amount превышает обязательства с плавающей ставкой.
>
> Когда указана `coin`, но `amount` не указан, применяются следующие правила в зависимости от `repaymentType`:
>
> - `FLEXIBLE`: Погашает только обязательства с плавающей ставкой по данной монете.
> - `FIXED`: Погашает только обязательства с фиксированной ставкой по данной монете.
> - `ALL`: Погашает все обязательства по данной монете (сначала с плавающей ставкой, затем с фиксированной).

> Важное замечание:
>
> - При погашении система будет использовать только доступный спотовый баланс валюты долга. Пользователи могут
>   выполнить погашение вручную, не конвертируя другие активы.
> - Чтобы проверить доступную спотовую сумму для погашения, вы можете вызвать этот эндпоинт
>   [Получить доступную сумму для погашения SPOT MARGIN TRADE](#)
> - Погашение запрещено с 04:00 до 05:30. Проценты начисляются на основе суммы займа в 05:00.
> - Система сначала погашает обязательства с плавающей ставкой, а затем — с фиксированной ставкой.
> - Поведение API в зависимости от repaymentType:
>   - `FIXED`: асинхронный (asynchronous)
>   - `FLEXIBLE`: синхронный (synchronous)
>   - `ALL`: асинхронный (asynchronous)

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/account/no-convert-repay HTTP/1.1
  Host: api.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1675842997277
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  
  {
      "coin":"BTC",
      "amount":"0.01"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/no-convert-repay"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "coin": "BTC",
      "amount": "0.01"
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр           | Обязательный  | Тип    | Комментарии                                                                                                                                                                                                                                                                                       | По умолчанию |
|--------------------|---------------|--------|----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- ---|--------------|
| coin               | да            | string | ***Название монеты.***<br><br>- Только заглавными буквами                                                                                                                                                                                                                                         | -            |
| amount             | нет           | string | ***Сумма возврата.***<br><br>- Если `coin` не передана во входном параметре, `amount` не может быть передана во входном параметре                                                                                                                                                                 | -            |
| repaymentType      | нет           | string | ***Тип погашения.***<br><br>- `ALL` — погасить оба типа обязательств: сначала с плавающей ставкой (floating-rate), затем с фиксированной (fixed-rate)<br>- `FIXED` — погасить только обязательства с фиксированной ставкой<br>- `FLEXIBLE` — погасить только обязательства с плавающей ставкой    | `FLEXIBLE`   |
