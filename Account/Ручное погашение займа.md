## Информация

Запрос на ручное погашение займа (можно указать количество)

> Информация:
>
> - Если во входном параметре не переданы ни `coin`, ни `amount`, то погасить все обязательства.
> - Если во входном параметре передана `coin`, а `amount` не передана, то монета будет погашена полностью.
<!-- -->
> Важно:
>
> - Механизм погашения
>
>   - При погашении задолженности система сначала использует доступный спотовый баланс в валюте долга. Если этих
>     средств недостаточно, оставшаяся сумма будет погашена путём конвертации других активов в соответствии с
>     [очерёдностью ликвидации](https://www.bybit.com/en/announcement-info/fullstock-leverage-uta/).
>   - Если вы хотите погасить задолженность только за счёт спотового баланса и не запускать конвертацию валют,
>     воспользуйтесь функцией
>     [Ручное погашение займа без конвертации активов](<Ручное погашение займа без конвертации активов.md>).
>
> - Временные ограничения и начисление процентов
>
>   - Погашение запрещено в период с 04:00 до 05:30 каждого часа.
>   - Проценты начисляются на основе суммы займа (BorrowAmount) в 05:00 каждого часа.
>
> - Очерёдность погашения
>
>   - Система сначала погашает обязательства с плавающей ставкой (floating-rate), затем — с фиксированной (fixed-rate).
>
> - Изменения с 2026 года
>
>   - Начиная с 17 марта 2026 года (постепенное внедрение, полностью доступно с 24 марта 2026 года) для погашения можно
>     будет использовать BYUSDT.
>
> - Ограничения по MNT
>
>   - Использование MNT для погашения временно приостановлено. Погашение обязательств в MNT через функцию конвертации
>     (convert-repay) не поддерживается.
>   - Однако вы по-прежнему можете использовать
>     [Ручное погашение займа без конвертации активов](<Ручное погашение займа без конвертации активов.md>),
>     чтобы погасить MNT из текущего баланса.
>
> - Обновление механизма ручного погашения UTA Loan (с 10 февраля 2026)\
>   Начиная с 10 февраля 2026 года, 08:00 UTC, в механизм ручного погашения UTA Loan вносятся следующие изменения:
>
>   - Комиссия за конвертацию валюты при погашении будет рассчитываться с использованием наибольшей ставки комиссии
>     (из двух: ставка для залогового актива или для актива-долга).
>   - Вводится лимит на конвертацию валюты в одной транзакции — 300 000 USD (общая сумма конвертируемых средств не
>     должна превышать эквивалент 300 000 USD). Это сделано для повышения стабильности и усиления контроля за рисками.
>   - Подробнее см. в объявлении: [UTA Loan manual repayment update](https://announcements.bybit.com/en/article/uta-loan-manual-repayment-update-bltbef3f1ad72a8295d/).

## Конечная точка

`/v5/account/repay`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/account/repay HTTP/1.1
  Host: api.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1675842997277
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  
  {
      "coin":"BTC",
      "amount":"0.01"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/repay"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "coin": "BTC",
      "amount": "0.01"
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр    | Обязательный  | Тип    | Комментарии                                                                                                                         | По умолчанию |
|-------------|---------------|--------|-------------------------------------------------------------------------------------------------------------------------------------|--------------|
| coin        | нет           | string | ***Название монеты.***<br><br>- Только заглавными буквами                                                                           | -            |
| amount      | нет           | string | ***Сумма возврата.***<br><br>- Если `coin` не передана во входном параметре, `amount` не может быть передана во входном параметре.  | -            |
