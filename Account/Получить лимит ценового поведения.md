## Информация

Запрос конфигурации, определяющей поведение системы в случае, когда цена вашего лимитного ордера превышает лучшую цену
спроса (highest bid) или лучшую цену предложения (lowest ask). Ответ также содержит информацию о том, включён ли
режим [Delta Neutral mode](<Настройка режима Delta Neutral Mode.md>).

> Информация:
>
> Где:
>
> - x% - [priceLimitRatioX](</Market/Получить информацию об инструментах.md>)
> - y% - [priceLimitRatioY](</Market/Получить информацию об инструментах.md>)
>
> Для спота (Spot):
>
> - Макс. цена покупки: Min[Max(Index Price, Index Price × (1 + x%) + 2-Minute Average Premium), Index Price × (1 + y%)]
> - Мин. цена продажи: Max[Min(Index Price, Index Price × (1 – x%) + 2-Minute Average Premium), Index Price × (1 – y%)]
>
> Для фьючерсов (Futures):
>
> - Макс. цена покупки: Min[Max(Index Price, Mark Price × (1 + x%)), Mark Price × (1 + y%)]
> - Мин. цена продажи: Max[Min(Index Price, Mark Price × (1 – x%)), Mark Price × (1 – y%)]
<!-- -->
> Настройка по умолчанию:
>
> - Спот: `lpaSpot` = `false`.
>   - Если цена ордера превышает границу, система отклоняет запрос.
>
> - Фьючерсы: `lpaPerp` = `false`.
>   - Если цена ордера превышает границу, система автоматически корректирует цену до ближайшего допустимого значения
>     (т. е. максимальной цены спроса или минимальной цены предложения).

## Конечная точка

`/v5/account/user-setting-config`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/account/user-setting-config HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1753255927950
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 52
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/user-setting-config"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {}

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-ответа"></a>

## Параметры ответа

| Параметр     | Тип     | Комментарии                                                                                                                                                                                          |
|--------------|---------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| lpaSpot      | boolean | `true` - Если цена ордера превышает границу, система автоматически скорректирует цену до ближайшего допустимого значения<br>`false` - Если цена ордера превышает границу, система отклоняет запрос   |
| lpaPerp      | boolean | `true` - Если цена ордера превышает границу, система отклоняет запрос<br>`false` - Если цена ордера превышает границу, система автоматически скорректирует цену до ближайшего допустимого значения   |
| deltaEnable  | boolean | `true` - Если включен режим Delta Neutral Mode<br>`false` - Если выключен режим Delta Neutral Mode                                                                                                   |
