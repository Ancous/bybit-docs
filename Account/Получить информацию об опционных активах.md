## Информация

Запрос информации о прибылях и убытках по опционным активам для каждой монеты в рамках учетной записи.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/account/option-asset-info`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/account/option-asset-info HTTP/1.1
  Host: api.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1773230920000
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests
  
  from urllib.parse import urlencode
  
  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/option-asset-info"
  complete_request = base_url + end_point
  
  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"
  
  data = {}
  
  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }
  
  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-ответа"></a>

## Параметры ответа

| Параметр   | Тип    | Комментарии                                                                                          |
|------------|--------|------------------------------------------------------------------------------------------------------|
| result     | array  | ***Список информации о P&L по активам.***                                                            |
| coin       | string | ***Название монеты.***                                                                               |
| totalDelta | string | ***Суммарная дельта.***<br><br>- Включает только дельту от опционных позиций                         |
| totalRPL   | string | ***Суммарный реализованный P&L.***                                                                   |
| totalUPL   | string | ***Суммарный нереализованный P&L.***                                                                 |
| assetIM    | string | ***Начальная маржа по активу.***<br><br>- Включает IM, занятую под открытые ордера по опционам       |
| assetMM    | string | ***Поддерживающая маржа по активу.***<br><br>- Включает MM, занятую под открытые ордера по опционам  |
| sendTime   | number | ***Метка времени снимка данных snapshot (в миллисекундах).***                                        |
