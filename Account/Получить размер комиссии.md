## Информация

Запрос на размер торговой комиссии.

> Зона применения:  
>
> `spot` - спот  
> `option` - опцион  
> `linear` - контракт (расчет в USDT, USDC, ...)
>
> - `Perpetual` - контракт без даты экспирации
> - `Futures` - контракт с датой экспирации
>
> `inverse` - контракт (расчет в BTC, ETH, ...)
>
> - `Perpetual` - контракт без даты экспирации
> - `Futures` - контракт с датой экспирации

## Конечная точка

`/v5/account/fee-rate`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/account/fee-rate?symbol=ETHUSDT HTTP/1.1
  Host: api.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1676360412362
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/fee-rate"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "symbol": "ETHUSDT",
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.get_fee_rates(
      symbol="ETHUSDT",
  ))
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|[category](</19.Определения значений в запросах и ответах.md#category>)	|да           |string    |***Тип продукта.***<br><br>- `spot`, `linear`, `inverse`, `option`  |-   |
|[symbol](</19.Определения значений в запросах и ответах.md#symbol>)	    |нет          |string    |***Имя символа,***<br><br>- Только заглавными буквами<br>- Действительно для `spot`, `linear`, `inverse`.    |-   |
|baseCoin                     |нет  |string     |***Базовая монета***<br><br>- Только заглавные буквы.<br>- Действительно для `option`       |-   |

<a id="параметры-ответа"></a>

## Параметры ответа

|Параметр  |Тип       |Комментарии                                             |
|----------|----------|--------------------------------------------------------|
|category   |string      |Тип продукта `spot`, `option`.<br>В разделе `linear` и `inverse` этого поля нет.             |
|list           |array   |Массив объектов                   |
|symbol   |string      |Название символа.<br>Сохраняет `""` для параметров.                                       |
|baseCoin   |string      |Базовая монета<br>Для `linear` и `inverse` это поле отсутствует<br>Сохраняет `""` для `spot`.                                             |
|takerFeeRate   |string      |Ставка комиссии тейкера                                             |
|makerFeeRate   |string      |Ставка комиссии мейкера                                             |
