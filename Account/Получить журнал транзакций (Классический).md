<a id="информация"></a>

Запрос журналов транзакций в кошельке деривативов (классический счет) и счете обратных деривативов (обновленном до UTA)

>Зона применения:  
>
>`spot` - спот  
>`option` - опцион  
>`linear` - контракт (расчет в USDT, USDC, ...)
>
> - `Perpetual` - контракт без даты экспирации
> - `Futures` - контракт с датой экспирации
>
>`inverse` - контракт (расчет в BTC, ETH, ...)
>
> - `Perpetual` - контракт без даты экспирации
> - `Futures` - контракт с датой экспирации

<a id="конечная-точка"></a>

## Конечная точка

`/v5/account/contract-transaction-log`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/account/contract-transaction-log?limit=1&symbol=BTCUSD HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1714035117255
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/contract-transaction-log"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
    "limit": 1,
    "symbol": "BTCUSD"
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|currency                     |нет  |string     |***Валюта актива.***<br><br>- Только заглавными буквами        |-   |
|baseCoin                     |нет  |string     |***Базовая монета.***<br><br>- Только заглавными буквами         |-   |

|startTime                     |нет  |integer     |***Временная метка начала выборки (в миллисекундах).***<br><br>- **Время начала и время окончания не передаются**:<br>&nbsp;&nbsp;&nbsp;по умолчанию возвращается 24 часа.<br>- **Передается только время начала**:<br>&nbsp;&nbsp;&nbsp;возвращается диапазон от времени начала + 24 часа.<br>- **Передается только время окончания**:<br>&nbsp;&nbsp;&nbsp;возвращается диапазон от времени окончания - 24 часа.<br>- **Если передаются оба значения**:<br>&nbsp;&nbsp;&nbsp;время окончания - время начала <= 7 дней.       |-   |
|endTime                     |нет  |integer     |***Временная метка конца выборки (в миллисекундах)***         |- |
|limit             |нет      	 |integer   |***Ограничение размера данных на странице.***<br><br>- [`1`, `50`]                     |`20`    |
|cursor	    |нет     |string   |***Курсор.***<br><br>- Используйте значение `nextPageCursor` из ответа для получения следующей страницы набора результатов   |-      |

## Параметры ответа

|Параметр  |Тип       |Комментарии                                             |
|----------|----------|--------------------------------------------------------|
|list   |array      |***Массив объектов.***                                             |
|id      |string     |***Уникальный идентификатор записи.***                            |
|symbol   |string      |***Название инструмента.***                                    |
|category   |string      |***Тип продукта.***                                             |
|side  |string       |***Направление сделки.***<br><br>- `Buy`<br>- `Sell`<br>- `None`      |
|transactionTime   |string      |***Временная метка транзакции (в миллисекундах).***     |
|[type](</19.Определения значений в запросах и ответах.md#type(contract-translog)>)    |string      |***Типы журналов транзакций.***        |
|qty   |string      |***Количество<br><br>- `Perpetual` и `Futures`: это количество для каждой сделки, не имеющее направления.                                             |
|size   |string      |***Размер.***<br><br>- Размер оставшейся позиции после исполнения сделки, указан её направление, короткая, со знаком `-` |
|currency   |string      |***Валюта.***<br><br>- `USDC`,`USDT`,`BTC`,`ETH` и т.д.                                                  |
|tradePrice   |string      |***Цена сделки.***                                             |
|funding   |string      |***Комиссия за финансирование.***<br><br>- Положительное значение означает вычет комиссии за финансирование<br>Отрицательное значение означает получение комиссии за финансирование.     |
|fee   |string      |***Торговая комиссия.***<br><br>- Положительное значение комиссии означает расходы<br>- Отрицательное значение комиссии означает пополнение |
|cashFlow   |string      |***Денежный поток.***<br><br>- Закрытие позиции и конвертация unRPL в RPL<br>- Перевод на счет или вывод.<br>Сюда не входят торговые комиссии и комиссии за финансирование.                  |
|change   |string      |`change` = `cashFlow` - `funding` - `fee`                                             |
|cashBalance   |string      |***Денежный баланс.***<br><br>- Это баланс кошелька после изменения наличных                                             |
|feeRate   |string      |***Ставка торговой комиссии.***<br><br>- Когда `type`=`TRADE`, это ставка комиссии за торговлю<br>- Когда `type`=`SETTLEMENT`, это ставка комиссии за финансирование.<br>- Для `side`=`Buy`, `feeRate` это положительная ставка рыночной комиссии<br>- Для `side`=`Sell`, `feeRate` это отрицательная ставка рыночной комиссии                                             |
|bonusChange   |string      |***Изменение бонуса.***                                             |
|tradeId   |string      |***Идентификатор сделки.***                                             |
|orderId   |string      |***Идентификатор ордера.***                                             |
|orderLinkId   |string      |***Пользовательский идентификатор ордера.***                                             |
|nextPageCursor  |string    |***Метка для следующей страницы.***<br><br>- Используется в параметре запроса `cursor` для пагинации          |
