<a id="информация"></a>

Запрос на получение информации об обеспечении по текущему единому маржинальному счету, включая процентную ставку по
кредиту, сумму кредита, коэффициент конвертации обеспечения, возможность использования залога в качестве маржи и т. д.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/account/collateral-info`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/account/collateral-info?currency=BTC HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1672127952719
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/account/collateral-info"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  api_secret = "<api_secret от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "currency": "BTC",
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<api_secret от биржи bybit>",
  )
  print(session.get_collateral_info(
      currency="BTC",
  ))
  ```

<a id="параметры-ответа"></a>

## Параметры ответа

|Параметр   |Тип       |Комментарии                                             |
|-----------|----------|--------------------------------------------------------|
|list       |array      |Массив объектов                                             |
|currency   |string      |Валюта всего текущего обеспечения                                             |
|hourlyBorrowRate   |string      |Почасовая ставка по займу                                             |
|maxBorrowingAmount   |string      |Максимальная сумма займа.<br>Это значение является общим для основных и дополнительных UID.       |
|freeBorrowingLimit   |string      |Максимальный лимит беспроцентного займа<br>- Максимальный лимит беспроцентного займа Беспроцентное заимствование возможно только по займам, возникшим в результате нереализованных убытков по контрактам.<br>- Спотовые маржинальные займы всегда облагаются процентами.                                          |
|freeBorrowAmount   |string      |Сумма займа в пределах общей суммы займа, освобожденная от уплаты процентов            |
|borrowAmount   |string      |Общая сумма займа                                             |
|otherBorrowAmount   |string      |Сумма заимствования по другим счетам в рамках одного основного счета                      |
|freeBorrowingAmount   |string      |**Устарело.** Всегда возвращает "", см. `freeBorrowingLimit`                                             |
|availableToBorrow   |string      |Доступная сумма для займа. Это значение является общим для основных и дополнительных идентификаторов.     |
|borrowable   |boolean      |Можно ли взять валюту в долг                                             |
|borrowUsageRate   |string      |Коэффициент использования займа: сумма основного и дополнительных счетов loanAmount/maxBorrowingAmount, это фактическое значение, 0,5 означает 50%                                             |
|marginCollateral   |boolean      |Можно ли использовать его в качестве валюты маржинального обеспечения<br>Если `marginCollateral`=`false`, то `collateralSwitch` не имеет смысла.                                               |
|collateralSwitch   |boolean      |Включено ли обеспечение пользователем<br>Если `marginCollateral`=`true`, то `collateralSwitch` имеет смысл.                                             |
|collateralRatio   |string      |В связи с новой логикой расчета стоимости многоуровневого залога это поле перестанет быть точным с 19 февраля 2025 года. См. раздел [Получить коэффициент многоуровневого залога SPOT MARGIN TRADE](#).                                             |
