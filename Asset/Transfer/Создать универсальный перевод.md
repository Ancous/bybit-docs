<a id="информация"></a>

Запрос на создание универсального перевода между sub-account-sub-account или master-account-sub-account.

>Совет:
>
>- Используйте API-ключ master-account или sub-account для запроса.
>   - Для использования API-ключа sub-account он должен иметь разрешение `SubMemberTransferList`.
>   - При использовании API-ключа sub-account перевод возможен только на master-account.
>- Если вы столкнулись с ошибкой errorCode: 131228 и сообщением your balance is not enough (недостаточно средств),
> перейдите в раздел [Получить баланс одной монеты](<../Balances/Получить баланс отдельной монеты.md>), чтобы
> проверить безопасную сумму для перевода.
>- Вы не можете осуществлять переводы между одинаковыми UID.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/transfer/universal-transfer`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/asset/transfer/universal-transfer HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1672189449697
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  
  {
      "transferId": "be7a2462-1138-4e27-80b1-62653f24925e",
      "coin": "ETH",
      "amount": "0.5",
      "fromMemberId": 592334,
      "toMemberId": 691355,
      "fromAccountType": "CONTRACT",
      "toAccountType": "UNIFIED"
  
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/transfer/universal-transfer"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  api_secret = "<api_secret от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "transferId": "be7a2462-1138-4e27-80b1-62653f24925e",
      "coin": "ETH",
      "amount": "0.5",
      "fromMemberId": 592334,
      "toMemberId": 691355,
      "fromAccountType": "CONTRACT",
      "toAccountType": "UNIFIED",
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<api_secret от биржи bybit>",
  )
  print(session.create_universal_transfer(
      transferId="be7a2462-1138-4e27-80b1-62653f24925e",
      coin="ETH",
      amount="0.5",
      fromMemberId=592334,
      toMemberId=691355,
      fromAccountType="CONTRACT",
      toAccountType="UNIFIED",
  ))
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|transferId                     |да            |string  |***[UUID](https://www.uuidgenerator.net/dev-corner)***<br><br>Пожалуйста, сгенерируйте UUID вручную.       |-   |
|coin                     |да            |string  |***Базовая монета.***<br><br>Только заглавными буквами        |-   |
|amount                     |да            |string  |***Количество***       |-   |
|fromMemberId                     |да            |integer  |***UID учетной записи откуда будет производиться перевод***       |-   |
|toMemberId                     |да            |integer  |***UID учетной записи куда будет производиться перевод***       |-   |
|[fromAccountType](<../../19.Определения значений в запросах и ответах.md#accountType>)   |да   |string  |***Тип учетной записи откуда будет производиться перевод***  |-   |
|[toAccountType](<../../19.Определения значений в запросах и ответах.md#accountType>)   |да   |string  |***Тип учетной записи куда будет производиться перевод*** |-   |
