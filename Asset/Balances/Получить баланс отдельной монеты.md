<a id="информация"></a>

Запрос баланса конкретной монеты для определенного типа аккаунта. Поддерживает запросы баланса для sub-account с
использованием их UID. Кроме того, можно проверить сумму, доступную для перевода между master-account и sub-account
(в обе стороны), а также между sub-account, что особенно актуально для пользователей с институциональным кредитом.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/transfer/query-account-coin-balance`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/asset/transfer/query-account-coin-balance?accountType=UNIFIED&coin=USDT&toAccountType=FUND&withLtvTransferSafeAmount=1 HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1690254520644
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/transfer/query-account-coin-balance"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  api_secret = "<api_secret от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "accountType": "UNIFIED",
      "coin": "BTC",
      "memberId": 592324,
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<api_secret от биржи bybit>",
  )
  print(session.get_coin_balance(
      accountType="UNIFIED",
      coin="BTC",
      memberId=592324,
  ))
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|memberId                     |нет           |string  |***UID***.<br><br>Требуется при запросе баланса sub-UID с помощью ключа API master-UID.       |-   |
|toMemberId                   |нет           |string  |***UID***.<br><br>Требуется при запросе баланса, который можно перевести между разными UID.       |-   |
|[accountType](<../../19.Определения значений в запросах и ответах.md#accountType>)     |да  |string    |***Тип аккаунта***     |-   |
|[toAccountType](<../../19.Определения значений в запросах и ответах.md#accountType>)   |нет |string    |***Тип аккаунта***.<br><br>Требуется при запросе переводимого баланса между счетами разных типов.       |-   |
|coin                         |да            |string  |***Базовая монета.***<br><br>Только заглавными буквами       |По умолчанию|
|withBonus                    |нет           |integer |`0`- не бонус запроса<br>`1`- бонус запроса       |`0`|
|withTransferSafeAmount       |нет           |integer |******Запросить ли отложенную сумму вывода/перевода******<br><br>- `0`- false, `1`- true<br>- Что такое [отложенная сумма вывода](#)?      |`0`  |
|withLtvTransferSafeAmount    |нет           |integer |***Для пользователей, которые берут кредиты через OTC, Вы можете узнать, какую сумму можно перевести, в зависимости от уровня риска***<br><br>- `0`- false, `1`- true<br>Тип счета обязателен       |`0`  |
