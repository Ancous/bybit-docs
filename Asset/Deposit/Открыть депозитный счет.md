<a id="информация"></a>

Запрос на установление счета для автоматического перевода после внесения депозита. Аналогичная функция настройке
депозита в [веб-интерфейсе](https://www.bybit.com/app/user/settings).

>Информация:
>
>- Ваши средства будут зачислены на кошелёк FUND по умолчанию. Вы можете настроить автоматический перевод средств после
> пополнения с помощью этого API.
>- Доступ возможен только для основного UID.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/deposit/deposit-to-account`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/asset/deposit/deposit-to-account HTTP/1.1
  Host: api.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1676887913670
  X-BAPI-RECV-WINDOW: 50000
  Content-Type: application/json
  
  {
      "accountType": "CONTRACT"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/deposit/deposit-to-account"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "accountType": "CONTRACT",
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.set_deposit_account(
      accountType="CONTRACT",
  ))
  ```
