<a id="информация"></a>

Запрос записей о депозитах в рамках платформы Bybit. Эти транзакции не находятся в блокчейне.

>Правила:
>
>- Максимальная разница между `startTime` - `endTime` составляет 30 дней.
>- Поддержка получения записей о депозитах по API-ключу master-account или sub-account.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/deposit/query-internal-record`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/asset/deposit/query-internal-record HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1682099024473
  X-BAPI-RECV-WINDOW: 50000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/deposit/query-internal-record"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  api_secret = "<api_secret от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "startTime": 1667260800000,
      "endTime": 1667347200000,
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<api_secret от биржи bybit>",
  )
  print(session.get_internal_deposit_records(
      startTime=1667260800000,
      endTime=1667347200000,
  ))
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

|Параметр    |Обязательный	 |Тип  	  |Комментарии                                                     |По умолчанию|
|------------|---------------|--------|----------------------------------------------------------------|------------|
|txID        |нет   |string  |***Идентификатор транзакции внутреннего перевода***             |-   |
|startTime   |нет   |integer |***Временная метка начала выборки (в миллисекундах)***<br><br>Значение по умолчанию: за 30 дней до текущего времени.   |-   |
|endTime  	 |нет   |integer |***Временная метка конца выборки (в миллисекундах)***<br><br>Значение по умолчанию: текущее время.      |-   |
|coin        |нет   |string  |***Название монеты***<br><br>Только заглавными буквами<br>Значение по умолчанию: все.       |-   |
|limit       |нет   |integer |***Ограничение размера данных на странице.***<br><br>[`1`, `50`]          |`50`  |
|cursor	     |нет   |string  |***Курсор.***<br><br>- Используйте значение `nextPageCursor` из ответа для получения следующей страницы набора результатов   |-      |

<a id="параметры-ответа"></a>

## Параметры ответа

|Параметр   |Тип       |Комментарии                                             |
|-----------|----------|--------------------------------------------------------|
|rows       |array     |Массив объектов                                             |
|id   |string    |Уникальный идентификатор                                             |
|type   |integer    |`1`: Внутренний депозит                                             |
|coin       |string    |Название монеты                                             |
|amount     |string    |Сумма пополнения                                             |
|status |string   |Статус депозита<br>`1`: Обработка<br>`2`: Успех<br>`3`: Ошибка депозита     |
|address   |string    |Адрес электронной почты или номер телефона                                             |
|createdTime   |string    |Временная метка создания депозита                                             |
|txID       |integer   |Идентификатор транзакции внутреннего перевода                                             |
|taxDepositRecordsId   |string    |Это поле используется в налоговых целях пользователями Bybit EU (Австрия). Укажите свой налоговый идентификатор.                                          |
|taxStatus   |integer   |Это поле используется для налоговых целей пользователями Bybit EU (Австрия)<br>`0`: Отчетность не требуется<br>`1`: Отчетность ожидается<br>`2`: Отчетность завершена                                         |
|nextPageCursor  |string    |Метка для следующей страницы (используется в параметре запроса `cursor` для пагинации)          |
