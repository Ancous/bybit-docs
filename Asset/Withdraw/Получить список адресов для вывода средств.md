## Информация

Запрос адреса для вывода средств из адресной книги

> Важное замечание:
>
> Ключ API для запроса этой конечной точки должен иметь разрешения на вывод средств.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/withdraw/query-address`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/asset/withdraw/query-address HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1672194949557
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/withdraw/query-address"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {}

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр    | Обязательный  | Тип     | Комментарии                                                                                                                                                                                                                                                                                 | По умолчанию |
|-------------|---------------|---------|---------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| coin        | нет           | string  | ***Название монеты.***<br><br>- Когда передается `coin`=`baseCoin`, это относится к универсальным адресам<br>- Когда передается название монеты, это относится к обычному адресу в сети                                                                                                     | -            |
| chain       | нет           | string  | ***Название сети.***<br><br>- Когда передается только название сети, возвращаются как обычные адреса, так и универсальные адреса<br>- Когда передается название сети и `coin`=`baseCoin`, возвращается только универсальный адрес, соответствующий сети                                     | -            |
| addressType | нет           | integer | ***Тип адреса.***<br><br>- `0`: Адрес в блокчейне (обычный тип адреса и универсальный тип адреса)<br>- `1`: Адрес для внутреннего перевода (недопустимые параметры `coin` и `chain`)<br>- `2`: Адрес в блокчейне и адрес для внутреннего перевода (недопустимые параметры `coin` и `chain`) | -            |
| limit       | нет           | integer | ***Ограничение размера данных на странице.***<br><br>- [`1`, `50`]                                                                                                                                                                                                                          | `50`         |
| cursor      | нет           | string  | ***Курсор.***<br><br>- Используйте значение `nextPageCursor` из ответа для получения следующей страницы набора результатов                                                                                                                                                                  | -            |

## Параметры ответа

| Параметр       | Тип     | Комментарии                                                                                                                                                                                                 |
|----------------|---------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| rows           | array   | ***Массив объектов.***                                                                                                                                                                                      |
| coin           | string  | ***Название монеты.***                                                                                                                                                                                      |
| chain          | string  | ***Сеть.***                                                                                                                                                                                                 |
| address        | string  | ***Адрес.***                                                                                                                                                                                                |
| tag            | string  | ***Тег адреса.***                                                                                                                                                                                           |
| remark         | string  | ***Замечание.***                                                                                                                                                                                            |
| status         | integer | ***Статус адреса.***<br><br>- `0`: Обычный<br>- `1`: Новым адресам запрещено выводить монеты в течение 24 часов                                                                                             |
| addressType    | integer | ***Тип адреса.***<br><br>- `0`: Тип адреса OnChain (обычный тип адреса и универсальный тип адреса)<br>- `1`: Тип адреса внутреннего перевода<br>- `2`: Тип адреса внутреннего перевода и тип адреса OnChain |
| verified       | integer | ***Подтверждён ли адрес.***<br><br>- `0`: Неподтверждённый адрес<br>`- 1`: Подтверждённый адрес                                                                                                             |
| createAt       | string  | ***Время создания адреса.***                                                                                                                                                                                |
| nextPageCursor | string  | ***Метка для следующей страницы.***<br><br>- Используется в параметре запроса `cursor` для пагинации                                                                                                        |
