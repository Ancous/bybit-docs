## Информация

Запрос записи об обмене монет.

>Информация:
>
>Иногда задержка составляет 5 секунд.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/exchange/order-record`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/asset/exchange/order-record?limit=10 HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1672990462492
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/exchange/order-record"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "limit": 10
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.get_coin_exchange_records(
      limit=10,
  ))
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр | Обязательный  | Тип     | Комментарии                                                                                                                | По умолчанию |
|----------|---------------|---------|----------------------------------------------------------------------------------------------------------------------------|--------------|
| fromCoin | нет           | string  | ***Валюта, из которой производится конвертация.***<br><br>Только заглавные буквы                                           | -            |
| toCoin   | нет           | string  | ***Валюта, в которую производится конвертация.***<br><br>Только заглавные буквы                                            | -            |
| limit    | нет           | integer | ***Ограничение размера данных на странице.***<br><br>[`1`, `50`]                                                           | `10`         |
| cursor   | нет           | string  | ***Курсор.***<br><br>- Используйте значение `nextPageCursor` из ответа для получения следующей страницы набора результатов | -            |

| Параметр       | Тип    | Комментарии                                                                                          |

| toAmount       | string | ***Сумма для конвертации в.***                                                                       |
| exchangeRate   | string | ***Курс конвертации.***                                                                              |
| createdTime    | string | ***Время создания конвертации (в секундах).***                                                       |
| exchangeTxId   | string | ***Идентификатор конвертации.***                                                                     |
