<a id="информация"></a>

Запрос подтверждения конвертации

>Информация:
>
>- Обмен асинхронный; проверьте окончательный статус, вызвав API результата запроса.
>- Убедитесь, что вы подтвердили предложение до истечения срока его действия.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/exchange/convert-execute`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/asset/exchange/convert-execute HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1720071899789
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 52
  
  {
      "quoteTxId": "10100108106409343501030232064"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/exchange/convert-execute"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  api_secret = "<api_secret от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "quoteTxId": "10100108106409343501030232064",
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<api_secret от биржи bybit>",
  )
  print(session.confirm_a_quote(
      quoteTxId="10100108106409343501030232064",
  ))
  ```
