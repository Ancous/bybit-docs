## Примеры запроса

- HTTP

  ```http
  POST /v5/asset/exchange/quote-apply HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1720071077014
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 172
  
  {
      "requestId": "test-00002",
      "fromCoin": "ETH",
      "toCoin": "BTC",
      "accountType": "eb_convert_funding",
      "requestCoin": "ETH",
      "requestAmount": "0.1",
      "paramType": "opFrom",
      "paramValue": "broker-id-001"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/exchange/quote-apply"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "requestId": "test-00002",
      "fromCoin": "ETH",
      "toCoin": "BTC",
      "accountType": "eb_convert_funding",
      "requestCoin": "ETH",
      "requestAmount": "0.1",
      "paramType": "opFrom",
      "paramValue": "broker-id-001",
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|[accountType](</19.Определения значений в запросах и ответах.md#convertAccountType>)    |да  |string     |***Тип кошелька***    |-   |
|fromCoin                     |да  |string     |***Монета, из которой конвертировать (монета для продажи)***       |-   |
|toCoin                     |да  |string     |***Монета, в которую конвертировать (монета для покупки)***       |-   |
|requestCoin                     |да  |string     |***Запрашиваемая монета, совпадает с `fromCoin`***<br><br>- В будущем мы можем поддерживать `requestCoin`=`toCoin`(для запроса на покупку напрямую)       |-   |
|requestAmount                     |да  |string     |***Количество запрашиваемой монеты (сумма, которую вы хотите продать)***       |-   |
|fromCoinType                     |нет  |string     |***Тип fromCoin***<br><br>`crypto`       |-   |
|toCoinType                     |нет  |string     |***Тип toCoin***<br><br>`crypto`       |-   |
|paramType                     |нет  |string     |`opFrom`<br>В основном используется для пользователей API-брокеров       |-   |
|paramValue                     |нет  |string     |***Идентификатор брокера***<br><br>В основном используемый для пользователей API-брокеров       |-   |
|requestId                     |нет  |string     |***Индивидуальный идентификатор запроса***<br><br>- Максимальная длина 36 символов<br>- Обычно это бесполезно, но удобно для внутреннего отслеживания запроса, если вы заполните это поле.       |-   |

<a id="параметры-ответа"></a>

## Параметры ответа

|Параметр  |Тип       |Комментарии                                             |
|----------|----------|--------------------------------------------------------|
|quoteTxId   |string      |Идентификатор конвертации. Он генерируется системой и используется для подтверждения котировки и запроса результата транзакции.        |
|exchangeRate   |string      |Курс обмена                                             |
|fromCoin   |string      |Монета, из которой конвертируется. (Совпадает с запросом.)                                             |
|fromCoinType   |string      |Тип монеты из которой конвертируется. (обычно `"crypto"`).                                             |
|toCoin   |string      |Монета, в которую конвертируется. (Совпадает с запросом.)                                             |
|toCoinType   |string      |Тип монеты в которую конвертируется. (обычно `"crypto"`).                                             |
|fromAmount   |string      |Количество монет из которой конвертируется. (сумма для продажи, как в запросе)                       |
|toAmount   |string      |Количество монет в которую конвертируется. (сумма для покупки по обменному курсу)                |
|expiredTime   |string      |Время истечения этого предложения (15 секунд).                                              |
|requestId   |string      |Индивидуальный идентификатор запроса                                             |
|extTaxAndFee   |array      |Поле, связанное с комплаенсом (соответствием правилам). Сейчас возвращает пустой массив, но может использоваться в будущем                                             |
