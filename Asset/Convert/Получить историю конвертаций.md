<a id="информация"></a>

Запрос на получения всех подтверждённых предложений конвертации.

>Информация:
>
>Отображает только историю конвертации, созданных через API.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/exchange/query-convert-history`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/asset/exchange/query-convert-history?accountType=eb_convert_uta,eb_convert_funding HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1720074159814
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/exchange/query-convert-history"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "accountType": "eb_convert_uta,eb_convert_funding",
      "coin": "BTC",
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP

  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.get_convert_history(
      accountType="eb_convert_uta,eb_convert_funding",
  ))
  ```

## Параметры запроса

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|[accountType](</19.Определения значений в запросах и ответах.md#convertAccountType>)   |нет  |string     |***Тип кошелька***<br><br>- Поддерживает передачу нескольких типов, разделенных запятой, например: `"eb_convert_funding, eb_convert_uta"`<br>- Возвращает данные всех типов кошельков, если параметр не передан       |-   |
|index                     |нет  |integer     |***Номер страницы***      |`1`   |
|limit                     |нет  |integer     |***Количество записей на страницы***<br><br>- Поддерживается до 100 записей на страницы<br>- Возвращает 100 записей, если количество записей превышает 100       |`20`   |

<a id="параметры-ответа"></a>

## Параметры ответа

|Параметр  |Тип       |Комментарии                                             |
|----------|----------|--------------------------------------------------------|
|list   |array      |Массив объектов                                             |
|[accountType](</19.Определения значений в запросах и ответах.md#convertAccountType>)   |string      |Тип кошелька    |
|exchangeTxId   |string      |Идентификатор конвертации                                             |
|userId   |string      |Идентификатор пользователя                                             |
|fromCoin   |string      |Монета, из которой конвертируется. (Совпадает с запросом.)                                             |
|fromCoinType   |string      |Тип монеты из которой конвертируется. (обычно `"crypto"`).                                             |
|toCoin   |string      |Монета, в которую конвертируется. (Совпадает с запросом.)                                             |
|toCoinType   |string      |Тип монеты в которую конвертируется. (обычно `"crypto"`).                                             |
|fromAmount   |string      |Количество монет из которой конвертируется. (сумма для продажи, как в запросе)                       |
|toAmount   |string      |Количество монет в которую конвертируется. (сумма для покупки по обменному курсу)                |
|extInfo   |object      |Объекты                                                                     |
|exchangeStatus   |string      |Статус обмена<br>`init`: инициализация<br>`processing`: в процессе<br>`success`: успешно<br>`failure`: неудача         |
|paramType   |string      |Это поле публикуется, когда вы отправляете его в запросе [Получить предложение конвертации](</Asset/Convert/Получить предложение конвертации.md>)            |
|paramValue   |string      |Это поле публикуется, когда вы отправляете его в запросе [Получить предложение конвертации](</Asset/Convert/Получить предложение конвертации.md>)                                       |
|convertRate   |string      |Курс обмена                                             |
|createdAt   |string      |Временная метка создания конвертации (в миллисекундах)   |
