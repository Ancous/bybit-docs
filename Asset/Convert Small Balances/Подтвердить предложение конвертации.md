## Информация

Запрос подтверждения конвертации

> Информация
>
> - Обмен выполняется асинхронно. Для проверки финального статуса операции используйте запрос
>   [Получить историю конвертаций](<Получить историю конвертаций.md>).
> - Убедитесь, что вы подтверждаете котировку до истечения её срока действия.
<!-- -->
> - Разрешение API-ключа: `Convert`
> - Лимит частоты запросов API: 5 запросов в секунду.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/covert/small-balance-execute`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/asset/covert/small-balance-execute HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1766128195297
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 49
  
  {
      "quoteId": "1010075157602517596339322880"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests
  
  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/covert/small-balance-execute"
  complete_request = base_url + end_point
  
  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"
  
  data = {
    "quoteId": "1010075157602517596339322880"
  }
  
  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }
  
  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)
  
  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP
  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.confirm_a_quote_small_balance(
      quoteId="1010075157602517596339322880",
  ))
  ```

<a id="параметры-ответа"></a>

## Параметры ответа

| Параметр     | Тип    | Комментарии                                                                                                                                                                  |
|--------------|--------|------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| quoteId      | string | ***Идентификатор котировки.***                                                                                                                                               |
| exchangeTxId | string | ***Идентификатор операции обмена.***<br><br>- Имеет то же значение, что и `quoteId`                                                                                          |
| submitTime   | string | ***Метка времени подтверждения операции.***                                                                                                                                  |
| status       | string | ***Статус операции.***<br><br>- `init`: создана<br>- `processing`: в обработке<br>- `success`: успешно<br>- `failure`: ошибка<br>- `partial_fulfillment`: частично исполнена |
| msg          | string | ***Сообщение.***<br><br>- По умолчанию - `""`                                                                                                                                |
