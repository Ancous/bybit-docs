## Информация

Запрос списка монет с небольшим балансом, которые можно конвертировать.

>Информация:
>
>Проверяйте наличие криптовалют с небольшим балансом, эквивалентным менее чем 10 USDT, и убедитесь, что общая сумма
>каждой конвертационной транзакции находится в диапазоне от 0,000000010 до 200 USDT.
<!-- -->
>- Разрешение ключа API: `Convert`  
>- Лимит частоты запросов API: 10 запросов в секунду

<a id="конечная-точка"></a>

## Конечная точка

`/v5/asset/covert/small-balance-list`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/asset/covert/small-balance-list?fromCoin=XRP&accountType=eb_convert_uta HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1766125546001
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests
  
  from urllib.parse import urlencode
  
  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/covert/small-balance-list"
  complete_request = base_url + end_point
  
  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"
  
  data = {
      "fromCoin": "XRP",
      "accountType": "eb_convert_uta",
  }
  
  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }
  
  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP
  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.get_small_balance_coins(
      fromCoin="XRP",
      accountType="eb_convert_uta",
  ))
  ```

## Параметры запроса

|Параметр                     |Обязательный  |Тип     |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|accountType                     |да  |string     |***Тип кошелька.***<br><br>- `eb_convert_uta`<br>- Поддерживается только Единый кошелёк (Unified wallet)    |-   |
|fromCoin                     |нет  |string     |***Исходная валюта.***       |-   |

|exchangeRate   |string       |**Игнорировать.**<br><br>- Зарезервированное поле                                             |
|feeInfo   |null       |**Игнорировать.**<br><br>- Зарезервированное поле                                             |
|taxFeeInfo   |null       |**Игнорировать.**<br><br>- Зарезервированное поле                                             |
