## Информация

Запрос предложения конвертации

>Информация:
>
>Конвертация недоступна для кастодиальных аккаунтов (например, Copper, Fireblocks и т.д.).  
>
>В едином торговом счете (Unified Trading Account) фактически исполненная сумма может быть меньше вашего доступного
>баланса. Если вы отправляете несколько запросов на конвертацию для разных криптовалют одновременно, возможны частичные
>исполнения. Сверяйтесь с фактически зачисленными суммами.
<!-- -->
>- Разрешение API-ключа: `Convert`  
>- Лимит частоты запросов API: 5 запросов в секунду.

## Конечная точка

`/v5/asset/covert/get-quote`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/asset/covert/get-quote HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1766126592271
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 97

  {
      "accountType": "eb_convert_uta",
      "fromCoinList": ["XRP", "SOL"],
      "toCoin": "USDC"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests
  
  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/asset/covert/get-quote"
  complete_request = base_url + end_point
  
  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"
  
  data = {
    "accountType": "eb_convert_uta",
    "fromCoinList": ["XRP", "SOL"],
    "toCoin": "USDC"
  }
  
  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }
  
  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)
  
  print(response.json())
  ```

- pybit

  ```python
  from pybit.unified_trading import HTTP
  session = HTTP(
      testnet=True,
      api_key="<api_key от биржи bybit>",
      api_secret="<secret_key от биржи bybit>",
  )
  print(session.request_a_quote_small_balance(
      accountType="eb_convert_uta",
      fromCoinList=["XRP", "SOL"],
      toCoin="USDC",
  ))
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр     | Обязательный | Тип    | Комментарии                                                                                                                 | По умолчанию |
|--------------|--------------|--------|-----------------------------------------------------------------------------------------------------------------------------|--------------|
| accountType  | да           | string | ***Тип кошелька.***<br><br>- `eb_convert_uta`<br>- Поддерживается только Единый кошелёк (Unified wallet)                    | -            |
| fromCoinList | да           | array  | ***Исходная валюта.***<br><br>- ["BTC", "XRP", "ETH"]<br>- За одну транзакцию можно указать до 20 монет                     | -            |
| toCoin       | да           | string | ***Валюта, в которую производится конвертация.***<br><br>- Каждый запрос поддерживает одну из следующих: MNT, USDT или USDC | -            |
