<a id="информация"></a>

Запрос результата конвертации, отправив `tradeNo` или `merchantRequestId`.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/fiat/trade-query`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  GET /v5/fiat/trade-query?tradeNo=TradeNo123456 HTTP/1.1  
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1720074159814
  X-BAPI-RECV-WINDOW: 5000
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import requests

  from urllib.parse import urlencode

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/fiat/trade-query"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "tradeNo": "TradeNo123456",
  }

  param_str = time_stamp + api_key + recv_window + urlencode(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.get(url=complete_request, headers=headers, params=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

|Параметр  	                  |Обязательный	 |Тип  	  |Комментарии       |По умолчанию|
|-----------------------------|--------------|--------|------------------|------------|
|tradeNo                     |нет  |string     |***Номер торгового ордера.***<br><br>Необходимо указать `tradeNo` или `merchantRequestId`.       |-   |
|merchantRequestId         |нет   |string     |***Индивидуальный идентификатор запроса***<br><br>Необходимо указать `tradeNo` или `merchantRequestId`.       |-   |

<a id="параметры-ответа"></a>

## Параметры ответа

|Параметр  |Тип       |Комментарии                                             |
|----------|----------|--------------------------------------------------------|
|result   |object      |Массив объектов                                             |
|tradeNo   |object      |Номер торгового ордера                                             |
|status   |string      |Статус обмена<br>`processing`: в процессе<br>`success`: успешно<br>`failed`: неудача         |
|quoteTxId   |string      |Идентификатор конвертации. Он генерируется системой и используется для подтверждения котировки и запроса результата транзакции.      |
|exchangeRate   |string      |Курс обмена                                            |
|fromCoin   |string      |Монета, из которой конвертируется. (монета на продажу.)                                             |
|fromCoinType   |string      |Тип монеты из которой конвертируется. (`fiat` или `"crypto"`).                         |
|toCoin   |string      |Монета, в которую конвертируется. (монета на покупку.)                                             |
|toCoinType   |string      |Тип монеты в которую конвертируется. (`fiat` или `"crypto"`).                             |
|fromAmount   |string      |Количество монет из которой конвертируется. (сумма для продажи, как в запросе)           |
|toAmount   |string      |Количество монет в которую конвертируется. (сумма для покупки по обменному курсу)       |
|createdAt     |number    |Временная метка создания ордера (в миллисекундах)                 |
|subUserId   |string      |Дочерний идентификатор пользователя в bybit                                             |
