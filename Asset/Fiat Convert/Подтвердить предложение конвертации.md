## Информация

Запрос подтверждения конвертации

>Информация:
>
>- Обмен асинхронный; проверьте окончательный статус, вызвав API результата запроса.
>- Убедитесь, что вы подтвердили предложение до истечения срока его действия.

<a id="конечная-точка"></a>

## Конечная точка

`/v5/fiat/trade-execute`

<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/fiat/trade-execute HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1720071899789
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 52
  
  {
      "quoteTxId": "QuoteTaxId123456",
      "subUserId":"43456"
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/fiat/trade-execute"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "quoteTxId": "QuoteTaxId123456",
      "subUserId": "43456"
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр          | Обязательный  | Тип    | Комментарии                                                                                                                                                                                            | По умолчанию |
|-------------------|---------------|--------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------|
| quoteTxId         | да            | string | ***Идентификатор конвертации.***<br><br>- Поле `quoteTxId` из [Получить предложение конвертации](</Asset/Fiat Convert/Получить предложение конвертации.md>)                                            | -            |
| subUserId         | да            | string | ***Дочерний идентификатор пользователя в bybit.***                                                                                                                                                     | -            |
| webhookUrl        | нет           | string | ***Ваш URL-адрес API для вызова в случае успешного или неудачного заказа.***<br><br>- URL нужно указывать вместе с конечной точкой (`"https://api.yourdomain.com/webhook"`)<br>- Максимум 256 символов | -            |
| MerchantRequestId | нет           | string | ***Индивидуальный идентификатор запроса.***<br><br>- Максимальная длина 36 символов<br>- Обычно это бесполезно, но удобно для внутреннего отслеживания запроса, если вы заполните это поле             | -            |
