<a id="примеры-запроса"></a>

## Примеры запроса

- HTTP

  ```http
  POST /v5/fiat/quote-apply HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1720071077014
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 172
  
  {
      "fromCoin": "ETH",
      "fromCoinType": "fiat",
      "toCoin": "BTC",
      "toCoinType": "crypto",
      "requestAmount": "0.1",
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests

  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/fiat/quote-apply"
  complete_request = base_url + end_point

  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"

  data = {
      "fromCoin": "ETH",
      "fromCoinType": "fiat",
      "toCoin": "BTC",
      "toCoinType": "crypto",
      "requestAmount": "0.1",
  }

  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }

  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)

  print(response.json())
  ```

<a id="параметры-запроса"></a>

## Параметры запроса

| Параметр         | Обязательный  | Тип    | Комментарии                                                               | По умолчанию |
|------------------|---------------|--------|---------------------------------------------------------------------------|--------------|
| fromCoin         | да            | string | ***Монета, из которой конвертировать (монета для продажи).***             | -            |
| fromCoinType     | да            | string | ***Тип fromCoin.***<br><br>- `crypto`<br>- `fiat`                         | -            |
| toCoin           | да            | string | ***Монета, в которую конвертировать (монета для покупки).***              | -            |
| toCoinType       | да            | string | ***Тип toCoin.***<br><br>- `crypto`<br>- `fiat`                           | -            |
| requestAmount    | да            | string | ***Количество запрашиваемой монеты (сумма, которую вы хотите продать).*** | -            |
| requestCoinType  | нет           | string | ***Тип монеты для продажи.***<br><br>- `fiat`, `crypto`                   | `fiat`       |

## Параметры ответа

| Параметр     | Тип    | Комментарии                                                                                                                                   |
|--------------|--------|-----------------------------------------------------------------------------------------------------------------------------------------------|
| quoteTxId    | string | ***Идентификатор конвертации.***<br><br>- Он генерируется системой и используется для подтверждения котировки и запроса результата транзакции |
| exchangeRate | string | ***Курс обмена.***                                                                                                                            |
| fromCoin     | string | ***Монета, из которой конвертируется.***<br><br>- Монета на продажу                                                                           |
| fromCoinType | string | ***Тип монеты из которой конвертируется.***<br><br>- `fiat`, `"crypto"`                                                                       |
| toCoin       | string | ***Монета, в которую конвертируется.***<br><br>- Монета на покупку                                                                            |
| toCoinType   | string | ***Тип монеты в которую конвертируется.***<br><br>- `fiat`, `"crypto"`                                                                        |
| fromAmount   | string | ***Количество монет из которой конвертируется.***<br><br>- Сумма для продажи, как в запросе                                                   |
| toAmount     | string | ***Количество монет в которую конвертируется.***<br><br>- Сумма для покупки по обменному курсу                                                |
| expiredTime  | string | ***Время истечения этого предложения.***<br><br>- 15 секунд                                                                                   |
