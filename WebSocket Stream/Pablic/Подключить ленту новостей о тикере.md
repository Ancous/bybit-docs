<a id="информация"></a>

Подписка на ленту новостей о тикере.

***Частота обновлений***

- `spot` — 50 мс.
- `perpetual`, `futures` и `option` — 100 мс

>Примечание:
>
>- В этой теме используются поля `snapshot` и `delta`. Если параметр ответа не найден в сообщении, его значение не
> изменилось.
>- Сообщения о тикерах `spot` и `option` контрактов содержат только `snapshot`.

## Linear/Inverse

<a id="примеры-подписки1"></a>

### Примеры подписки

- собственная реализация

  ```python
  import json
  import time
  import websocket
  
  URL = "wss://stream-testnet.bybit.com/v5/public/linear"
  
  def on_message(ws, message):
      print(f"Полученное сообщение: {message}")
  
  def on_error(ws, error):
      print(f"Ошибка: {error}")
  
  def on_close(ws, close_status_code, close_msg):
      print(f"Соединение закрыто: {close_status_code} {close_msg}")
  
  def on_open(ws):
      print("Соединение открыто")
      request = {
          "op": "subscribe",
          "args": [
              "tickers.ETHUSDT"
          ]
      }
      ws.send(json.dumps(request))
  
  def run_websocket():
      while True:
          ws = websocket.WebSocketApp(URL)
          ws.on_message = on_message
          ws.on_error = on_error
          ws.on_close = on_close
          ws.on_open = on_open
  
          ws.run_forever(
              ping_interval=20,
              ping_timeout=5
          )
  
          print("Соединение потеряно. Попытка переподключения...")
          time.sleep(5)
  
  run_websocket()
  ```

- pybit

  ```python
  from pybit.unified_trading import WebSocket

  from time import sleep
  ws = WebSocket(
      testnet=True,
      channel_type="linear",
  )
  def handle_message(message):
      print(message)
  ws.ticker_stream(
      symbol="BTCUSDT",
      callback=handle_message
  )
  while True:
      sleep(1)
  ```

### Параметры ответа

|Параметр   |Тип         |Комментарии                                             |
|-----------|------------|--------------------------------------------------------|
|topic | string | Название потока данных. |
|type | string | Тип данных: `snapshot`, `delta`. |
|cs | integer | Кросс-последовательность. |
|ts | number | Метка времени (в миллисекундах), когда система генерирует данные. |
|data | array | Массив объектов. |
|symbol | string | Название символа. |
|[tickDirection](<../../19.Определения значений в запросах и ответах.md#tickDirection>) | string | Направление тика. |
|price24hPcnt | string | Процентное изменение цены за последние 24 часа. |
|lastPrice | string | Последняя цена. |
|prevPrice24h | string | Цена 24 часа назад. |
|highPrice24h | string | Максимальная цена за последние 24 часа. |
|lowPrice24h | string | Минимальная цена за последние 24 часа. |
|prevPrice1h | string | Цена 1 час назад. |
|markPrice | string | Mark Price. |
|indexPrice | string | Index Price. |
|openInterest | string | Размер открытого интереса. |
|openInterestValue | string | Стоимость открытого интереса. |
|turnover24h | string | Оборот за последние 24 часа. |
|volume24h | string | Объём за последние 24 часа. |
|nextFundingTime | string | Время следующего финансирования (в миллисекундах). |
|fundingRate | string | Ставка финансирования. |
|bid1Price | string | Лучшая цена покупки. |
|bid1Size | string | Объём лучшей заявки на покупку. |
|ask1Price | string | Лучшая цена продажи. |
|ask1Size | string | Объём лучшей заявки на продажу. |
|deliveryTime | datetime | Время поставки (timestamp в миллисекундах), актуально только для фьючерсов с истечением срока. |
|basisRate | string | Базисная ставка. Уникальное поле для `inverse` и `linear` |
|deliveryFeeRate | string | Плата за экспирацию. Уникальное поле для `inverse` и `linear`. |
|predictedDeliveryPrice | string | Предполагаемая цена поставки. Уникальное поле для `inverse` и `linear`. |
|preOpenPrice | string | Оценочная цена открытия предварительного-контракта (неактуальна после открытия рынка)<br><br>- `inverse` и `linear` не имеют этого поля |
|preQty | string | Оценочный объём предварительного-контракта (неактуален после открытия рынка)<br><br>- `inverse` и `linear` не имеют этого поля |
|[curPreListingPhase](<../../19.Определения значений в запросах и ответах.md#curPreListingPhase>) | string | Текущая предварительная фаза контракта<br><br>- `inverse` и `linear` не имеют этого поля |
|fundingIntervalHour | string | Интервал финансирования в часах<br><br>- Сейчас поддерживается только целое число часов<br>- Только для `perpetuals`, для `futures` это поле не возвращается |
|fundingCap | string | Верхний и нижний пределы ставки финансирования<br><br>- Только для `perpetuals`, для `futures` это поле не возвращается |
|basisRateYear | string | Годовая базисная ставка<br><br>- Только для `futures`, для `perpetuals` это поле не возвращается |
