<a id="информация"></a>

Подписка на свечи

>Важное замечание:
>
>Если `confirm`=`true`, это означает, что свеча закрылась. В противном случае свеча всё ещё открыта и обновляется.

***Доступные intervals***

- `1` `3` `5` `15` `30` `60` `120` `240` `360` `720` (минут)
- `D` (день)
- `W` (неделя)
- `M` (месяц)

***Частота обновлений***

- 1-60 с.

<a id="поток-данных"></a>

## Поток данных

`kline.{interval}.{symbol}`

## Подписка

```json
{
    "op": "subscribe",
    "args": [
        "kline.30.BTCUSDT"
    ]
}
```

<a id="примеры-подписки"></a>

## Примеры подписки

- собственная реализация

  ```python
  import json
  import time
  import websocket
  
  URL = "wss://stream-testnet.bybit.com/v5/public/linear"
  
  def on_message(ws, message):
      print(f"Полученное сообщение: {message}")
  
  def on_error(ws, error):
      print(f"Ошибка: {error}")
  
  def on_close(ws, close_status_code, close_msg):
      print(f"Соединение закрыто: {close_status_code} {close_msg}")
  
  def on_open(ws):
      print("Соединение открыто")
      request = {
          "op": "subscribe",
          "args": [
              "kline.5.BTCUSDT"
          ]
      }
      ws.send(json.dumps(request))
  
  def run_websocket():
      while True:
          ws = websocket.WebSocketApp(URL)
          ws.on_message = on_message
          ws.on_error = on_error
          ws.on_close = on_close
          ws.on_open = on_open
  
          ws.run_forever(
              ping_interval=20,
              ping_timeout=5
          )
  
          print("Соединение потеряно. Попытка переподключения...")
          time.sleep(5)
  
  run_websocket()
  ```

- pybit

  ```python
  from time import sleep

  from pybit.unified_trading import WebSocket

  ws = WebSocket(
      testnet=True,
      channel_type="linear",
  )

  def handle_message(message):
      print(message)

  ws.kline_stream(
      interval=5,
      symbol="BTCUSDT",
      callback=handle_message
  )

  while True:
      sleep(1)
  ```

<a id="параметры-получаемых-данных"></a>

## Параметры получаемых данных

|Параметр  |Тип       |Комментарии                                             |
|----------|----------|--------------------------------------------------------|
|topic     |string    |Название потока данных |
|type      |string    |Тип данных. `snapshot` |
|ts        |number    |Метка времени (в миллисекундах), когда система генерирует данные |
|data      |array     |Массив объектов |
|start     |number    |Начальная временная метка (в миллисекундах) |
|end       |number    |Конечная временная метка (в миллисекундах) |
|[interval](</19.Определения значений в запросах и ответах.md#interval>)   | string   |Интервал Kline |
|open      |string    |Цена открытия |
|close     |string    |Цена закрытия |
|high      |string    |Наивысшая цена |
|low       |string    |Наименьшая цена |
|volume    |string    |Объём торгов |
|turnover  |string    |Оборот |
|confirm   |boolean   |Закрыта ли свеча |
|timestamp |number    |Временная метка (в миллисекундах) последнего совпавшего ордера в свече |
