## Информация

Для начала торговли контрактами на сырьевые товары необходимо сначала подписать соглашение. После его подписания
вам станут доступны все контракты на сырьевые товары категории «металлы» (metals commodity contracts).

> Информация:
>
> Только master-UID может подписать соглашение через этот эндпоинт. Sub-UID не поддерживаются для данной операции.
> После того как master-UID подписал соглашение, все sub-UID получают право на торговлю.
> API-key master-UID должен обладать одним из следующих разрешений для вызова этой конечной точки:
>
> - “Account Transfer”
> - “Subaccount Transfer”
> - “Withdrawal”

## Конечная точка

`/v5/user/agreement`

## Примеры запроса

- HTTP

  ```http
  POST /v5/user/agreement HTTP/1.1
  Host: api-testnet.bybit.com
  X-BAPI-API-KEY: "<api_key от биржи bybit>"
  X-BAPI-SIGN: <подпись>
  X-BAPI-TIMESTAMP: 1772695036541
  X-BAPI-RECV-WINDOW: 5000
  Content-Type: application/json
  Content-Length: 40
  
  {
      "agree": true,
      "category": 2
  }
  ```

- собственная реализация

  ```python
  import time
  import hmac
  import hashlib
  import json
  import requests
  
  base_url = "https://api-testnet.bybit.com"
  end_point = "/v5/user/agreement"
  complete_request = base_url + end_point
  
  api_key = "<api_key от биржи bybit>"
  secret_key = "<secret_key от биржи bybit>"
  time_stamp = str(int(time.time() * 1000))
  recv_window = "5000"
  
  data = {
      "agree": true,
      "category": 2
  }
  
  param_str = time_stamp + api_key + recv_window + json.dumps(data)
  
  signature = hmac.new(
      key=secret_key.encode("utf-8"),
      msg=param_str.encode("utf-8"),
      digestmod=hashlib.sha256
  ).hexdigest()
  
  headers = {
    "X-BAPI-API-KEY": api_key,
    "X-BAPI-SIGN": signature,
    "X-BAPI-TIMESTAMP": time_stamp,
    "X-BAPI-RECV-WINDOW": recv_window,
  }
  
  response = requests.post(url=complete_request, headers=headers, json=data, timeout=10)
  
  print(response.json())
  ```

## Параметры запроса

| Параметр  | Обязательный  | Тип      | Комментарии                                                                | По умолчанию   |
|-----------|---------------|----------|----------------------------------------------------------------------------|----------------|
| category  | да            | integer  | ***Категория контракта***<br><br>- `2`: Контракты на сырьевые товары       | -              |
| agree     | да            | boolean  | `true` — согласие с условиями.                                             | -              |
